# Neutral placeholder brand assets

`header.bmp` (150×57) and `welcome.bmp` (164×314) are **brand-free** solid-colour
MUI2 bitmaps generated by `tools/gen_placeholder_assets.py`.

They exist so a plugin that ships no NSIS bitmaps (e.g. Chasm, which came from an
Inno Setup project and only carries `icon.png`) still compiles a valid installer.
CI copies these in only when the plugin's own `packaging/` lacks them.

**Do not replace these with another plugin's branded art.** Cross-plugin bitmap
bleed — Churn's installer describing Polygraph, etc. — is the class of bug this
whole project exists to eliminate. A plugin that wants custom installer art
should commit its own `header.bmp` / `welcome.bmp` to its repo's `packaging/`.

Regenerate:

```
python3 tools/gen_placeholder_assets.py templates/assets
```
