# InstallerGenerator

Central build + installer factory for DirektDSP audio plugins — one manifest per plugin (`plugins/*.toml`), unified installers (NSIS / macOS `.pkg` / Linux zip), and a `catalog.json` for a desktop bulk-installer.

The software available in this repo is defined as any file available through the releases section on this GitHub page.
